### ‚öôÔ∏è Run a project

> **Note**
> Create new unsplash API key and add client id and api key in `local.properties` as `CLIENT_ID="[your_client_id]"` and `BASE_URL="[https://api.unsplash.com/]"` then Sync project with gradle file to run it.
